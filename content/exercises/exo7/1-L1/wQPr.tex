\texte{
Soient $n,p \in \N^*$ avec $n < p!$. Montrer qu'il existe
un unique $p$-uplet $(n_1,\dots,n_p)$ d'entiers naturels tel que
$$\forall\ k \le p,\ n_k \le k,\quad\text{et}\quad
  n = \sum_{k=1}^p n_kk!.$$
}
}
