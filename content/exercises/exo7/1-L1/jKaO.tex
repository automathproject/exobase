\texte{
Soit $n \in \N$ fix{\'e}. D{\'e}terminer pour quelle valeur de $p$ le nombre
$C_n^p$ est maximal (on {\'e}tudiera le rapport $C_n^p/C_n^{p+1}$).
}
\reponse{
$p = \left[\frac {n+1}2 \right]$.
}
